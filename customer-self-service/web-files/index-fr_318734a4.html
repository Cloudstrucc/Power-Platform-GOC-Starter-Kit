Documentation

La BOEW fonctionne sur le principe d'amélioration progressive, et implémente des techniques bien établies pour accroître l'accessiblité et l'utilisabilité de contenu Web. Si vous êtes nouveau à la BOEW, veuillez commencer par notre section Comment démarrer, qui contient de l'information comment télécharger, utiliser ou contribuer à la boîte d'outils. Sinon, visualisez la section référence ci-bas.

Comment démarrer

  • Je veux utiliser la BOEW pour mon site Web
  • Je veux personaliser BOEW et créer un nouveau thème BOEW
  • Je veux signaler une anomalie ou demander une question à propos de la BOEW
  • Je veux corriger ou améliorer la BOEW en tant que développeur

Lignes directrices pour créer ou modifier la documentation

  • Verser tous les guides, les descriptions ou les directives pour la BOEW dans le répertoire de GitHub.
  • Suivre le modèle de documentation.
  • Fournir au moins un exemple pour chaque attribut, thème ou élément de la BOEW. D’autres exemples doivent être donnés pour chaque paramètre ou option.
  • Si vous avez des questions, considerer de rencontrer l'équipe de la BOEW durant un de nos code sprint hebdomadiare à chaque mardi.

Avis concernant la BOEW v3.1

À partir du 23 septembre 2014, la version 3.1 de la Boîte à outils de l'expérience Web n'est plus supportée. Le code source et la documentation ont été déplacés vers le dépôt wet-boew-legacy.
