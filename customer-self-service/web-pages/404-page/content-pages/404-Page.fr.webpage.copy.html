Nous sommes désolés, 404 non trouvé, 404, erreur 404, page non trouvée ou fichier non trouvé message d'erreur est un code de réponse standard du protocole de transfert hypertexte (HTTP), dans les communications de réseau informatique, pour indiquer que le navigateur a pu communiquer avec un serveur donné, mais le serveur n'a pas pu trouver ce qui était demandé. L'erreur peut également être utilisée lorsqu'un serveur ne souhaite pas révéler s'il dispose des informations demandées.
